# Family Cash Card Service

The Family Cash Card is an app for families to manage allowances in the form of digital debit cards.

---

## Project Features:
- **REST API Development:** Building a robust and scalable API with Spting Boot
- **Security:** Implement authentication and authorization layer
- **Quality assurance:** Practicing Test Driven Development (TDD) to maintain high-quality code
- **Data Management:** Handling automated persistence with Spring Data JDBC.

## Technology and principles:
- **Language**: Java
- **Framework:** Spring Boot (Security, Data JDBC)
- **Version Control:** Git & GitHub (Conventional Commits)
- **Best Practices:** Clean Code, and TDD.
